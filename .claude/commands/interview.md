---
description: Задать вопросы через AskUserQuestion (по одному)
---

Задавай открытые вопросы ко мне через инструмент `AskUserQuestion`, не текстом.

- **Один вопрос за вызов** — несколько вопросов задавай последовательно.
- **2-4 варианта** на вопрос; опция "Other" для свободного ответа добавляется сама.
- **Рекомендуемый вариант** — первым, с `(Recommended)` в конце label.
- **multiSelect: true** — только когда можно выбрать несколько.
- **preview** — только для визуального сравнения (код, мокап, конфиг).
- **Язык** — русский.
