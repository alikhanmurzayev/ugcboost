description: Загрузить все стандарты кода проекта UGCBoost
---

Прочитай все файлы в `docs/standards/` — это обязательные стандарты проекта (архитектура, именование, тесты,
безопасность и т.д.). Читай каждый файл полностью, без сокращений.

После прочтения отчитайся коротко:

- Сколько файлов прочитано
- Список по областям: backend / frontend / общие
- Готовность применять стандарты в текущей работе

Не пересказывай содержимое — только подтверди загрузку. К деталям обращайся по мере необходимости в ходе работы.
