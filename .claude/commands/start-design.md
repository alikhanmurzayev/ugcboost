---
description: Старт дизайн-сессии — prime + standards + spec-mode
---

Шорткат для запуска дизайн-сессии. Выполни последовательно три шага. После каждого — краткий отчёт о готовности, потом сразу следующий, без паузы на подтверждение.

1. **Prime.** Прочитай и выполни инструкции из `.claude/commands/prime.md`.
2. **Standards.** Прочитай и выполни инструкции из `.claude/commands/standards.md`.
3. **Spec mode.** Прочитай и выполни инструкции из `.claude/commands/spec-mode.md`. Аргумент команды (если был передан в `/start-design`) — это intent для spec-mode.

После шага 3 ты находишься в spec-mode RULES + INSTRUCTIONS — продолжаешь по ним.
